# blackbag [![Build Status](https://secure.travis-ci.org/michael-may/blackbag.png?branch=master)](http://travis-ci.org/michael-may/blackbag)

## Getting Started
Install the module with: `npm install blackbag`

blackbag.awesome(); // "awesome"
